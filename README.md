**My Portfolio Website (based on Terminus)**

This project is built using the [Terminus Astro Template](https://github.com/ojoanalogo/terminus-astro-template) but modified for my needs.

**Changes**
- Updated site configuration
- Added custom components
- Adjusted styling
- Deployed under my GitHub Pages setup

**Quick Start**
1. Clone the repo
2. Install dependencies with `npm install`
3. Run locally with `npm run dev`
4. Open `http://localhost:4321`
